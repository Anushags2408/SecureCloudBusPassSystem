# Secure Cloud Bus Pass System

## Overview

Secure Cloud Bus Pass System is a Flask-based web application that provides secure online bus ticket booking with QR-code ticket generation. The system includes encrypted user credentials, SQL injection prevention, and cloud deployment support.

## Features

* User Registration and Login
* Encrypted Password Storage
* SQL Injection Protection
* Online Bus Ticket Booking
* QR Code Ticket Generation
* SQLite Database
* Cloud Ready Deployment

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* QRCode Library
* Cryptography Library

## Project Modules

1. User Management
2. Security Module
3. Ticket Booking Module
4. QR Ticket Generation Module

## Security Features

* Password Encryption
